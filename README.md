# Resumo 
O Secredts-project é um projeto básico de Node.js onde o bodyParser e o req.body leem o valor que venho do input logo na sequência a função passwordCheck verifica a senha retornada usando o If Else. Caso a senha esteja de acordo irá renderizar na tela uma receita de sobremesa de chocolate caso contrário ira retorna para a tela inicial.

# instalação
1- Baixe todos os arquivos.
2- Abra ele no seu editor de código.
3- No terminal digite "npm i" para instalar todas as dependências.
4- Instalado as dependências digite no terminal "node index.js".
5- O projeto irá renderizar no link "http://localhost:3000".
